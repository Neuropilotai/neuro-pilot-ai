⚠️ This must match the OWNER_DEVICE_ID value in Railway environment variables.
Stores JWT as np_owner_jwt and device as np_owner_device in localStorage.
❓ How to find your Device ID
  • Go to Railway dashboard → Your project → Settings → Variables
  • Look for OWNER_DEVICE_ID environment variable
  • Copy that exact value and paste it in the Device ID field above
  • If it doesn't exist, create it with any secure string (e.g., "my-macbook-pro-2024")
Note: The Device ID you enter here must exactly match the OWNER_DEVICE_ID value in Railway, or owner API calls will fail with 403 errors.
